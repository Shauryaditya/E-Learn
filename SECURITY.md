## Reporting a Vulnerability

We take the security of this project seriously. If you have found a vulnerability, please report it immediately:

1. **Do not** open a public issue on GitHub.
2. Email the maintainers directly (add email here if applicable) or create a [Private Vulnerability Report](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) if enabled on the repository.
3. Provide as much information as possible, including:
   - Steps to reproduce
   - Affected versions
   - Potential impact

We will acknowledge your report within 48 hours and provide a timeline for a fix. We appreciate your efforts to responsibly disclose your findings.
